# Sayings Database

An offline-first Android application designed to let you easily load, organize, and browse custom databases of sayings, fortunes, aphorisms, tech tips, or quotes. This lets you create your own personalized "message of the day" (or "Saying of the Day") on your Android device with custom shortcuts, searchable browse page, and multiple databases!

## Features

- **Multi-Database Support**: Create, rename, delete, and switch between completely separate collections or categories of text built from text files you import.
- **Multiple Text File Import Options**:  Choose from single-line, blank-line separated, or *nix fortune cookie formats.
- **Robust CSV & JSON Import**: Import from raw CSV (with or without headers), standard nested/flat JSON arrays, newline-delimited JSON (NDJSON / JSON Lines), or mixed JSON stream content.
- **Flexible Field Mapping**: Map columns or keys dynamically from your CSV or JSON files to the sayings body with or without annotation.
- **Search & Filter**: Retrieve entries quickly by index # or filter with a case-insensitive, multi-string keyword search.
- **Saying of the Day (SotD)**: Keeps track of your daily message. Includes index adjustment and daily offset tuning to keep your daily message exactly where you want it.
- **Homescreen Shortcut**: Pin shortcuts on your home screen for any of your custom databases for instant access (opens to Saying of the Day).
